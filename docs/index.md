#  欢迎来到我的个人技术笔记站

这里是我在 **机器学习** 和 **嵌入式软件开发（尤其是电赛相关）** 领域的学习与实践记录。

本网站致力于整理复杂的知识点，以清晰、易懂的结构进行呈现，方便自己查阅，也希望能为有相似学习路径的朋友提供参考。

---

## 🗺️ 主要笔记分类

我的笔记内容主要围绕以下两个核心领域展开，你可以通过左侧的导航栏或点击下面的链接进入相应部分。

### 🤖 机器学习 (Machine Learning)

该部分主要记录深度学习框架 **PyTorch** 的学习过程和实践心得。

* **核心内容:** Tensor 操作、数据加载与预处理、神经网络模块 (`nn`) 的实现、以及使用 **TensorBoard** 进行模型可视化。
* **快速入口:** [进入 PyTorch 学习笔记](机器学习/Pytorch_Learning/nn.ipynb) (以 `nn` 模块为例)

### 💻 电赛软件 (Embedded Software)

该部分聚焦于电子设计竞赛及嵌入式开发所需的软件知识和技能，内容涵盖底层架构到实时系统。

* **核心内容:** **ARM Cortex-M 架构**、**FreeRTOS** 实时操作系统、**STM32 HAL 库**应用、**通信协议** (I2C, SPI, CAN) 以及基础的**电控算法**。
* **快速入口:** [进入 FreeRTOS 实时操作系统](电赛软件/RTOS/FreeRTOS/FreeRTOS.md) (以 FreeRTOS 简介为例)
